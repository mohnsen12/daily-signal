## Comment (first comment fra maker)

Hi HN! 👋

I built an autonomous AI agent that writes and publishes a daily newsletter every single night — with zero human intervention.

The pipeline:
1. 22:00 — Agent scans 50+ tech/AI sources (TechCrunch, The Verge, Ars Technica, HN, etc.)
2. 01:00 — Selects top 5-7 stories based on relevance, novelty, and impact
3. 03:00 — Translates and summarizes everything into Danish
4. 05:00 — Generates HTML newsletter, RSS feed, shareable social card
5. 06:00 — Publishes automatically to GitHub Pages

The agent ("Teddy 🐕") also:
- Runs quality checks (10-point checklist, must score 80+ before publishing)
- Tracks analytics and content quality trends
- Generates tweet variants for social promotion
- Manages a referral system

Why Danish? There are dozens of English AI newsletters (TLDR, The Neuron, Superhuman), but zero that curate specifically for the Danish/Nordish market. It's a niche that's underserved.

Tech stack: Node.js, OpenClaw (agent framework), cron automation, GitHub Pages for hosting. Total cost: $0/month.

I've been running this for 3 days now, publishing daily at 6 AM CET. Quality has improved from 55/100 to 100/100 on the internal quality scale.

The most interesting part for me has been watching the agent make editorial decisions autonomously — which stories to prioritize, how to frame headlines, what angle to take in analysis. It's surprisingly good at it.

Happy to answer questions about the architecture, the editorial pipeline, or how to set up something similar for other languages/niches.

Source code: https://github.com/mohnsen12/daily-signal
Newsletter: https://mohnsen12.github.io/daily-signal/
RSS: https://mohnsen12.github.io/daily-signal/rss.xml
